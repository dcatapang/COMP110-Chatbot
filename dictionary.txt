algorithm
An algorithm is a step by step list of instructions that will solve a problem if executed.
Here is an example of an algorithm that computes the area of a square: (1) ask for the length, (2) square the length of the side, (3) display the area
integer
An integer is a whole number value.
Here is an example of an integer: 42
double
A double is a decimal number value.
Here is an example of a double: 5.67
string
A string is a sequence of characters that is enclosed by quotation marks.
Here is an example of a string: "Hello, World!"
data type
A data type is a classification of an object in Python, which defines what values it can take and what operations can be performed on it.
Here is an example of a data type: Integer (1, 2, -72, etc.)
operator
An operator is a special token that represents computations like addition, multiplication, and division.
Here is an example of an operator: Addition (+)
modulus operator
A modulus operator is an operator that works on integers and yields the remainder of the first integer divided by the second integer. It is represented by %.
Here is an example: 7 % 2 = 1
variable
A variable is a name that refers to a value
Here is an example of a variable: x
